# Fro (Frocus OS)

Fro is a productive first WebOS.

## Tech Stack

- Tanstack Start
- React (w/ compiler)
- Tailwind CSS
- Zustand (w/ immer)
- React-RND
- Biome
- Shadcn

## Apps

1. Notes (Markdown, and html)
2. Settings (for Wallpaper change, theme change, virtual pets)
3. Browser (Iframe based)
4. Terminal (with commands like ls, open [app], add [app], calculations)
5. Calculator (Simple arithmetic calculations)
6. Calendar (Day, Month, and Year view)
7. Music Player (Play some musics)
8. Fro Store (Downloads some extra apps and games)

## Widgets

1. Weather (Shows the current weather through an API)
2. Clock (Shows the time)
3. Quote (Shows random quotes)
4. Github (Connect to your profile URL, gives you stats)
5. Focus (Shows your focus status)
